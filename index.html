Tab Navigation

As the order of elements in the page follows a linear procedure, the page can be easily navigated using the tab button.

Access Keys

Access keys have been added for every tab: Q for Dialer, C for Contact List, A for Add Contact, and H for Help. Access keys have also been added for each of the numbers on the dialer, with each key matching the number. Z is the access key for clear. It should be noted that the WAVE tool suggests against the use of access keys.

Aria Landmarks

Aria tags ahve been added to the site. The ARIA tags define the header as navigation, and each page body as main. They also state the name and number are required for the add contact page.

Aria Live

The entire page body that the tabs rest in has been wrapped in a div with a polite aria live tag.

Tables

The contacts page has been updated so that each button is a table header. There are 3 buttons per row. More will be added to the table as the functionality of the site is expanded upon.
